## Getting Started
To build the plugin to your Roblox plugins folder:

```bash
rojo build -o "PLUGINSFOLDER\CurveEditor.rbxlx"
```

To rebuild your plugin every time you make changes, add a --watch flag

```bash
rojo build --watch -o "PLUGINSFOLDER\CurveEditor.rbxlx"
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
